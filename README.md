
#### Street2shop dataset tools

http://www.tamaraberg.com/street2shop/

Get the dataset list:
```
bash get_street2shop.sh
```

Download the images (Python 2.7):
```
python download.py --urls photos/photos.txt
```
or  (Python 2.7)
```
python download_photos.py
```
or (Python 3.x)
```
python dl.py
```
